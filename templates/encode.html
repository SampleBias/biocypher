How Encoding Works

The encoding process converts your text message into a DNA sequence following these steps:

  1. Convert each character to its ASCII value
  2. Convert ASCII values to 8-bit binary
  3. Map binary pairs to DNA bases (A=00, T=01, C=10, G=11)

The resulting DNA sequence can only be decoded with the correct mapping key.
